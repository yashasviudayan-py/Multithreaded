rust-highperf-server

A high-performance multithreaded HTTP/HTTPS web server built in Rust.

Features

  • HTTP/1.1 and HTTP/2 via ALPN auto-negotiation
  • TLS via rustls (no OpenSSL)
  • JWT authentication (HMAC-SHA256)
  • SQLite CRUD API via sqlx
  • HTML templating via Tera
  • Secure signed session cookies
  • Prometheus-compatible /metrics endpoint
  • IP allowlist / blocklist
  • Per-request timeout enforcement
